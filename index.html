Vigenere (Javascript)

Description

Ce programme JavaScript chiffre et déchiffre les messages en utilisant le chiffrement Vigenère. La clé doit avoir au moins une lettre. La clé est insensible à la casse et les lettres non-ignorées. Lors du cryptage ou du décryptage, la casse est conservée et les lettres non-modifiées ne sont pas modifiées. Le chiffre de César est un cas particulier du chiffre de Vigenère avec une clé à une lettre.

Le Code Source est disponible ICI.
